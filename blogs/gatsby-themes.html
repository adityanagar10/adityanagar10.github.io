Setting up Resuable Gatsby Themes

A single Theme file for multiple websites

August 29, 2021 Aditya Nagar

While working on a senior developer’s code for the first time, I was so overwhelmed with the code and couldn’t really understand anything. Everything seemed like magic. So just like any normal human, I went to Google and found a detailed documentation from Gatsby which was really useful, but it was kind of outdated. Hence I decided to write thing blog post.

#1 Setup Yarn Workspace:

You’ll create two workspaces, gatsby-theme-events and site.

Create a folder:

Create an empty folder and give it a title of your wish, here I am using Gatsby-Theme-Tutorial

Create a package.json in this folder.

{
"private": true,
"workspaces": ["gatsby-theme-events", "site"]
}

Set up gatsby-theme-events and site

In the root directory, create two new folders gatsby-theme-events and site and create package.json in both.

Now you should have a similar folder structure:

.
├── gatsby-theme-events
│ └── package.json
├── site
│ └── package.json
└── package.json


In package.json of gatsby-theme-events, paste the following code:

{
"name": "gatsby-theme-events",
"version": "1.0.0",
"main": "index.js",
"license": "MIT"
}
