what is Blockchain?

A Blockchain is a DATABASE and a NETWORK

August 25, 2021 Aditya Nagar

A standard WebApp is quite simple as we have a client and a server, where a client contacts the server and vice versa. But with Blockchain, we have a decentralized structure (DAPP) , where there are various nodes instead of a single server, i.e. the data is distributed across every device connected to the blockchain

Blockchain is a Peer-2-Peer network of nodes that talk to one another. Each node gets a copy of the data shared across the blockchain. All this data is bundle in a “box” called Blocks, that is chained together to create the Public Ledger.

Blockchain formation. The main chain (black) consists of the longest series of blocks from the genesis block (green) to the current block. Orphan blocks (purple) exist outside of the main chain.(Source: Wikipedia)

The “blocks" are linked together using cryptography. Every block contain a Cryptographic Hash of the block, a Timestamp and Transaction Data. As blocks each contain information about the block previous to it, they form a chain, with each additional block reinforcing the ones before it, making it resilient to changes once data has been recorded. Timestamp proves that the transaction data existed when the block was published in order to get into its hash All the nodes work together, in order to make the data in the public ledger secure and unchanged
